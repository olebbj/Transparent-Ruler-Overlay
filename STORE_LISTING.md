# Chrome Web Store Listing Draft

## Name

Transparent Ruler Overlay

## Short description

Transparent px + cm ruler overlay for fast visual spacing checks on any webpage.

## Detailed description

Transparent Ruler Overlay adds a clean, non-intrusive ruler on top of any webpage so designers and QA can quickly verify spacing without opening DevTools.

### Highlights

- Pixel and centimeter scales in one ruler
- Horizontal and vertical orientation
- Transparent overlay with adjustable opacity
- Quick hotkeys:
  - Toggle ruler: `Alt+R` (Windows/Linux), `Option+R` (Mac)
  - Toggle orientation: `Alt+Shift+R` (Windows/Linux), `Option+Shift+R` (Mac)
- Tick density presets: Sparse / Normal / Dense
- Drag directly with mouse
- Snap to viewport edges and pin controls
- Centimeter calibration:
  - Bank card reference (85.60 mm)
  - Custom 5 cm reference
- Per-site saved position/orientation/pin/snap
- Global saved appearance + calibration

### Privacy

- No data collection
- No analytics
- No network requests
- Preferences stored locally in browser storage only
