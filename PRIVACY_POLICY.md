# Privacy Policy

Last updated: February 21, 2026

## Summary

Transparent Ruler Overlay does **not** collect, store, or transmit personal data.

## Data handling

- No analytics
- No tracking pixels
- No remote API calls
- No user account system

## Permissions used

- `activeTab`: applies the ruler only to the tab you are currently using.
- `scripting`: injects the overlay script into the active page on demand.
- `storage`: saves your preferences locally (opacity, density, color theme, calibration, and per-site ruler position/settings).

## Storage location

All settings are stored locally in your browser via `chrome.storage.local`.
No data is sent to external servers.

## Third-party links

The popup contains a donate link to Ko-fi. Opening that link takes you to an external website governed by its own privacy policy.

## Contact

For support or questions, use the support channel listed in the extension store page.
